同济大学2026春《编译原理》课程作业
基于Rust语言实现一个简单的类rust语言编译器，支持基本的语法分析、语义分析和代码生成。
目前实现情况：

- [x] 词法分析：完成了基本的词法分析器，能够识别关键字、标识符、数字和符号。
- [x] 语法分析：实现了一个CLR(1)表驱动语法分析器，能够解析基本的表达式和语句。
- [x] 语义分析：完成了AST->HIR->THIR的语义分析和类型检查流程。
- [x] 代码生成：实现了一个以LLVM IR为目标的代码生成器，支持直接输出.ll中间代码文件，并使用LLVM后端生成可执行文件，同时支持调用外部函数，使用LLVM连接到c静态库来实现输出。
